The Smart IoT Workspace Monitoring and Automation System is an intelligent embedded system designed to improve workspace comfort, safety, automation, and environmental monitoring using IoT technologies. The system is built around the ESP32 microcontroller, which acts as the main processing and communication unit responsible for reading sensor data, controlling actuators, and transmitting real-time information to the cloud dashboard.

The project integrates multiple analog and digital sensors to monitor different environmental conditions inside the workspace. Two ultrasonic sensors are used as a visitor counting system to detect people entering and leaving the area. A sound sensor continuously measures environmental noise levels and compares the readings with a calibrated baseline value to classify sound conditions into quiet, moderate, and noisy levels. According to the detected sound intensity, different LEDs are activated, while excessive noise triggers a servo motor response.

The system also includes a DHT11 temperature sensor for room temperature monitoring. When the temperature exceeds a predefined threshold, a cooling fan is activated automatically to improve environmental comfort. In addition, an LDR sensor is used for smart lighting control by adjusting LED brightness automatically using PWM signals according to surrounding light conditions. Air quality monitoring is implemented using the MQ135 gas sensor, which activates a buzzer alarm when abnormal gas levels are detected.

The ESP32 continuously processes sensor data locally while simultaneously sending important readings and actuator states to the Blynk IoT cloud dashboard through Wi-Fi communication. The dashboard enables real-time monitoring and remote control of the system using virtual pins and cloud widgets.

The project also implements fail-safe behavior and autonomous local operation, allowing the ESP32 to continue executing critical control logic even during temporary Wi-Fi communication failure. Overall, the system demonstrates the practical implementation of IoT technologies, embedded systems, cloud monitoring, and smart automation using a low-cost and energy-efficient architecture suitable for modern smart workspace environments.
